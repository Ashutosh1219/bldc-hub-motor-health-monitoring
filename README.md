# Hub Motor Health Monitoring Using BLDC Fault Analysis

## Overview
This project focuses on condition monitoring and fault detection in hub-mounted BLDC motors using simulation-driven fault modeling and statistical feature extraction techniques.

## Objective
To identify electrical and mechanical fault signatures in BLDC hub motors and evaluate condition monitoring strategies for reliability enhancement.

## System Description
- BLDC motor dynamic model
- Electrical fault modeling
- Mechanical imbalance simulation
- Sensor fault injection

## Methodology
- MATLAB/Simulink-based modeling
- Feature extraction (statistical + time-domain)
- Comparative analysis of fault signatures

## Key Focus Areas
- Early fault detection
- Reliability enhancement
- Feature sensitivity analysis
- Data-driven monitoring
